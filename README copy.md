# Foodi Restaurant Website

A responsive restaurant landing page built with Next.js and Tailwind-inspired styling, modeled after a modern food delivery website design.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the app in your browser:
   ```
   http://localhost:3000
   ```

## Tech Stack Used

- **Next.js** – React framework for server-rendered and statically generated pages
- **React** – UI library for component-based development
- **Tailwind CSS / PostCSS** – utility-first styling and build-time CSS processing
- **JavaScript** – scripting language for client logic
- **HTML / CSS** – markup and styling for the website layout

## Assumptions

- The project is a static landing page for a restaurant or food delivery brand.
- The hero section and menu showcase are the primary visual focus; detailed backend ordering logic is not included.
- Fonts are loaded from Google Fonts and are used to match the requested brand typography.
- The layout is intended for modern desktop and mobile browsers.
